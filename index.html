Introduction to GIT

GIT is an open source version control system that is used to track the changes made to our code.

Advantages of using GIT

  • Fast and efficient
  • Distributed version control
  • Enhanced collaboration
  • Compactibility
  • Data integrity

Configuring Git

git config --global user.name "Winifred Epetuku"

git config --global user.email "epetukuwinifred@gmail.com"

Basic Git Command

  • git init: used to initialize a git repository
  • git add . : used to stage changes made to our
  • git status : used to check the status of our code
  • git commit -m "commit message" : this is used to save the staged changes with a descriptive message
  • git remote add origin : this is used to connect our local repository to our remote repository on github
  • git push origin main : this is used to push your code to local repository to remote repository on github
